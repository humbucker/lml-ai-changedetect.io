# workspace Agent Rules

This file documents local project rules and agent configurations for this workspace.

## Agent skills

### Issue tracker

Issues and PRDs for this repo live as GitHub issues. We do not treat external PRs as a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Using standard canonical triage labels (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context repository layout. See `docs/agents/domain.md`.
